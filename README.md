# 🚀 Create and Manage Automated Processes by using Power Automate

## 📖 Overview

This repository documents my learning journey through the Microsoft Learn Applied Skills path:

**Create and Manage Automated Processes by using Power Automate**

The objective of this learning path is to build practical knowledge of designing, creating, testing and managing cloud flows using Microsoft Power Automate.

Rather than simply completing the modules, I am documenting my learning, practical exercises, key takeaways, and how these concepts can be applied to real-world business scenarios.

# 🎯 Learning Objectives

Throughout this learning path I will be covering:

- Creating automated cloud flows
- Understanding triggers and actions
- Working with conditions
- Configuring approvals
- Managing flow runs
- Testing cloud flows
- Troubleshooting failed flows
- Sharing and managing flows
- Power Automate best practices
